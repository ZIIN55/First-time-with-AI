### 🔗 ลิงก์ทดลองเล่น AI ของพวกเรา

[วางลิงก์ที่ก๊อปมาจาก Teachable Machine ตรงนี้ได้เลย]  

ธีมที่ 1: "ทายอารมณ์ทะลุจอ" (Emotion Classifier)

• Class ที่ต้องเทรน: หน้ายิ้ม (Happy) / หน้าบึ้ง (Sad) / หน้าตกใจ (Surprised)
• ความสนุก: เด็กๆ ชอบเล่นหูเล่นตากับกล้องหน้า iPad แข่งกันว่า AI ของใครจะแยกอารมณ์กวนๆ ของเพื่อนได้แม่นที่สุด

ธีมที่ 2: "ตรวจจับออฟฟิศซินโดรม" (Ergonomic Posture)

• Class ที่ต้องเทรน: นั่งหลังตรง (Good Posture) / นั่งหลังค่อมหรือก้มหัวเล่นมือถือ (Bad Posture)
• ความสนุก: ได้ขยับร่างกาย และเห็นประโยชน์ของ AI ในชีวิตจริง ว่าเอามาทำระบบเตือนคนนั่งทำงานได้

ธีมที่ 3: "นักสืบสิ่งของในห้องเรียน" (Classroom Objects)

• Class ที่ต้องเทรน: ปากกา / ยางลบ / บัตรนักเรียน (หรือของอื่นๆ ในกระเป๋า)
• ความสนุก: ต้องสลับกันเอาของมาวางใกล้-ไกลกล้อง เพื่อเทรนให้ AI จำวัตถุให้ได้แม่นยำ

ธีมที่ 4: "เป่ายิ้งฉุบระดับสากล" (Rock-Paper-Scissors)

• Class ที่ต้องเทรน: ค้อน / กรรไกร / กระดาษ
• ความสนุก: เป็นโจทย์คลาสสิกที่ทำง่ายที่สุด เห็นผลลัพธ์ไว เหมาะสำหรับกลุ่มที่อยากเน้นความชัวร์
